About

View current statistics

CapstoneViewer allows users to view all capstone projects done in Ashesi University with the ability to see the current statistics of data. The viewer can see the total number of capstone projects to date, women percentages and men percentages, and percentages of projects by department.

Get updated data

The database is updated by the administrator everytime a capstone project is accomplished. The viewer sees all projects that are up to date easily because projects are categorized according to the department.

View all details

All the details of the capstone project are provided. If the user wants to explore a certain project, the names of the implementer, the contacts of the implementer, and the objectives of the project are provided.
